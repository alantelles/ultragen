In this example, you can see how to serve static files. In practice, when deploying your application you will (and should) use a high performance web server like NGINX or Apache httpd to serve static files. However, you can use it while developing your application and you can also use the UltraGen way to serve files in your favor. UltraGen method to serve static files is an ordinary method. It means that you can execute some script before serving the file. This example show how to do this by creating a zip of the site photos.
